This website is designed to help these people, and real estate investors in general.

DATA AND WORK

Obtained data regarding price, crime, school scores, age of the properties, number of bedrooms etc, from zillow.com, trulia.com, city-data.com, via API and web scraping using the python package called beautiful soup. The investment classes are obtained via the k-means clustering methods, based on cash flow, appreciation, and crime.


Four Classes:

Black: low cash flow, low to medium appreciation, high crime. Red: medium cash flow, medium appreciation, low crime.
Green: high cash flow, medium appreciation, moderately high crime. Blue: medium cash flow, high appreciation, medium crime.
